REPORTES


El sistema cuenta con varios reportes que permiten analizar la información, pero consientes de que cada asociación requiere de reportes específicos para su administración. Se puede configurar los reportes que más se adecuen a sus necesidades.

De igual manera los formatos de impresión son adaptados a las necesidades del cliente.

El sistema se entrega con un reporteador, para que los usuarios generen los reportes de acuerdo a sus requerimientos. Cuenta con una base de datos abierta, de tal manera que puede hacer interfase o liga con el sistema contable de su elección. Todos los reportes pueden ser generados a diferentes formatos como son: XML, PDF, etc.
